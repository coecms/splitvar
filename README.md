Build Instructions:
-------------------

Pip install (into a virtual environment):

    pip install splitvar

or
    
    pip install git+https://github.com/coecms/splitvar

or (for development)

    git checkout git+https://github.com/coecms/splitvar
    cd splitvar
    pip install -e '.[dev]'
