Build Instructions:
-------------------

Pip install (into a virtual environment):

    pip install splitvar

or
    
    pip install git+https://github.com/aidanheerdegen/splitvar

or (for development)

    git checkout git+https://github.com/aidanheerdegen/splitvar
    cd splitvar
    pip install -e '.[dev]'
